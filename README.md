# NSMB Docker
A simple all-in-one solution for setting up the [NSMB Code Mod Template](https://github.com/MammaMiaTeam/NSMB-Code-Template).

## Requirements
- [Docker](https://docs.docker.com/desktop/)
- A copy of New Super Mario Bros. (USA region)
- NitroSDK v3.0
- Nitro System
